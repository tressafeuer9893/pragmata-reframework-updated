4. Navigate with mouse or arrow keys
5. Check boxes to enable features
6. Adjust sliders for values
7. Press **`Insert`** again to close

### How to Install Mods

1. Download any Pragmata mod (Lua script, texture, model)
2. Open REFramework folder in game directory: `reframework/`
3. Copy mod files to appropriate subfolder:
   - **Lua scripts** → `reframework/autorun/`
   - **Textures** → `reframework/textures/`
   - **Models** → `reframework/models/`
4. Launch game
5. Mods load automatically
6. Some mods require enabling in menu

### How to Adjust Graphics

1. Press **`Insert`** → **"Graphics"** tab
2. Adjust settings:
   - **FOV** — Drag slider to 90 or 100 (recommended)
   - **Motion Blur** — Uncheck to disable
   - **Chromatic Aberration** — Uncheck to disable
   - **LOD Distance** — Set to 150% for better visuals
3. Changes apply immediately
4. Some settings need restart (marked with ⚠️)

### How to Use Cheats

1. Press **`Insert`** → **"Cheats"** tab

   - **No Reload** — Skip reload animations
3. Works immediately
4. Disable before boss fights (some may break)

### How to Install Scripts from Community
